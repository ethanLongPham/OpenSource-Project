Set Up Introduction - Belle&Cozy Funiture E-commerce Website

First of all, download the "src_code_opensource" folder.

To run the web app on Raspberry Pi or Linux
  1. Open PuTTY and Connect to Your Raspberry Pi
    • Open PuTTY
    • Enter the IP address or hostname of your Raspberry Pi.
    • Click "Open" and log in with your username and password.
  2. Ensure Git is Installed: Before you can clone a repository,
    you need to make sure that Git is installed on your Raspberry Pi.
    Run the following command to check if Git is installed: git --version
    If Git is not installed, you can install it using: sudo apt update sudo apt install git
  3. Navigate to the Desired Directory: Navigate to the directory where you want to clone the repository.
    For example, if you want to clone it to your home directory,
    you can simply do: cd ~
  4. Clone the Repository: Use the git clone command followed by the URL of the GitHub repository
    you want to clone. Replace with the actual URL of the repository:
    git clone
    Our github repo: https://github.com/ethanLongPham/OpenSource-Project.git
  5. Access the repository and run this .sql file: login_bellncozy_db.sql,
    so that when using function on the website, information will be saved to database.
    Make sure the port of Maria DB is 3307, because I already modify the port of php file to 3307.
    Make sure you name your database the same as the .sql file
  6. Download and run Apache. Make sure the source code folder is in Apache document directory
    If Apache is on, then you can try to access the web from whatever device using the same
    wifi connection with the Raspberry Pi.
To run the web app on Window
  1. You can download XAMPP, as it easier to set up. It's already included Apache,
    PHP, MySQL, etc.
  2. On XAMPP interface, start the Apache and MySQL services. Go to web browser and
    search http://localhost/phpmyadmin/. You can take the SQL query from the .sql file
    and create a database here.
  3. Then can just start to run the web
